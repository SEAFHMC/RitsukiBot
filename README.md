## RitsukiBot：基于Nonebot的自用QQbot
建议先创建虚拟环境，以poetry为例：<br>
```python
poetry install
```
什么你没有poetry？
```python
pip install poetry
```
使用```poetry shell```进入虚拟环境，进入RitsukiBot目录，运行```python bot.py```<br><br>
目前支持的功能有： 
- 生成每日水位监测报告
- Pixiv榜单查询与根据pid返回图片
- yande.re涩图
- 应用更新(仅Admin有效)
- 通过Bot运行服务器指令(没什么卵用
- RSS订阅信息
- 高浓度对话（够味），感谢[AnimeThesaurus](https://github.com/Kyomotoi/AnimeThesaurus)
